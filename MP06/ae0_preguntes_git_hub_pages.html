5. Què és una arrow function? A quin ES apareix (quin any).
• És una funcion que serveix para simplificar les coses i t'estalvia l’escriure masses línies de codi. En 2015 ya que es cuando salio el ES6.
6. Què és un callback? Posa un exemple.
• Un callback (llamada de vuelta) és una funció que rep com a argument una altra funció i l'execució.
• Un exemple molt comú de callback és com funció escuchadora d'un esdeveniment.
function show Alert () {
alert ( ‘Alerta’);
}
button.addEventListener ( ‘clic’, showAlert);
En aquest exemple, showAlert és un callback. També podem escriure el callback com a funció anònima.
